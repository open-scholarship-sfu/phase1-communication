# Open Scholarship Framework Initiative - Communication guidelines repository

This repository contains visual identity files and communication templates for the SFU Open Scholarship Framework, including:

- Visual identity guidelines and assets
- Presentation templates
- Report templates
- Poster templates
- Other communication tools and resources

## About the SFU Open Scholarship Framework

This repository supports the [SFU Open Scholarship Framework](https://www.sfu.ca/research/performance-excellence/open-science.html), an initiative to promote open science and scholarship practices at Simon Fraser University.

## Related Repositories

This repository is part of the phase 1 development work. For project planning, documentation, and management, see the main [phase1-base](https://github.com/open-scholarship-sfu/phase1-base) repository.

If this work is replicated for phase 2, this repository will be forked to maintain continuity while allowing for new developments.
